# 폴 짝 (Fall Jjak)

## 게임 소개

「폴 짝」은 한 마리의 개구리가 끝없이 펼쳐진 오염된 연못 위를 연잎에서 연잎으로 점프해 나아가는 캐주얼 아케이드 게임입니다. 플레이어는 마우스와 키보드를 활용한 정교한 컨트롤로 점프의 거리·높이·방향을 조절하며, 변화무쌍한 연잎과 날씨, 다양한 적과 장애물을 극복해 최고 점수에 도전합니다.

---

## 게임 기본 정보

| 항목 | 내용 |
|------|------|
| 게임명 | 폴 짝 |
| 장르 | 아케이드 |
| 플랫폼 | 웹 |
| 시점 | 고정형 쿼터뷰 (Isometric) |
| 그래픽 | 3D 복셀(Voxel) + 빅 픽셀 도트 아트 |
| 플레이 인원 | 싱글 플레이 (확장 시 비동기 경쟁 지원) |
| 입력 장치 | 키보드 + 마우스 |
| 플레이 타임 | 1회 약 2~10분 (스코어 어택 기반) |

---

## 핵심 목표

- 오직 앞으로만 전진하며 연잎을 밟고 최대한 멀리 이동합니다.
- 정확한 착지(Yarr!)와 콤보 누적으로 점수 배율을 끌어올립니다.
- 다양한 연잎 기믹과 동적 날씨, 적의 방해를 극복해 자신의 최고 점수를 갱신합니다.

---

## 핵심 규칙

### 진행 규칙

- 플레이어는 뒤로 돌아갈 수 없으며, 무조건 앞(또는 대각선 앞) 방향으로만 점프할 수 있습니다.
- 새로운 연잎을 밟고 무사히 착지하는 순간 점수가 누적됩니다.
- 물에 빠지거나, 적·장애물에 부딪히거나, 함정 연잎을 밟으면 즉시 게임 오버됩니다.
- 게임 오버 시 최종 점수와 이동한 연잎 수, 최고 콤보, 잡아먹은 파리 수가 결과 화면에 표기됩니다.

### 점수 산정

| 판정 | 기준 | 기본 점수 / 효과 |
|------|------|------------------|
| Yarr! | 연잎 정중앙 안착 | +30점, 콤보 +1 |
| Great | 중앙에서 다소 벗어난 안착 | +20점, 콤보 +1 |
| Not bad.. | 가장자리에 간신히 안착 | +10점, 콤보 초기화 |
| Miss | 연잎을 벗어나 물에 빠짐 | 게임 오버 (생존 수영 버프 시 1회 복귀) |

### 콤보 시스템

- Yarr! 또는 Great 판정을 연속으로 받을 때 콤보 스택이 1씩 누적됩니다.
- 콤보 수치에 비례해 획득 점수에 배율 보너스가 적용됩니다. (5콤보 × 1.5배, 10콤보 × 2.0배, 20콤보 × 3.0배)
- Not bad.. 판정을 받거나 물에 빠지는 즉시 콤보가 0으로 초기화됩니다.
- 일정 콤보 도달 시 이펙트·카메라 진동·BGM 강도 증가로 시각·청각 피드백을 제공합니다.

### 동적 난이도

점수와 진행 거리가 일정 임계값을 넘을 때마다 다음 요소가 점진적으로 강해집니다.

- 연잎 간 거리 및 위치 무작위성 증가
- 특수 연잎(삭은/이동/회전/함정/탄성/점멸) 등장 빈도 증가
- 동적 날씨 변경 주기 단축
- 적(물고기·새) 출현 패턴 다양화

## 조작 방법

「폴 짝」은 단순한 한 번의 클릭이 아니라 **조준 → 거리 조절 → 궤적 조절 → 발사**라는 4단계의 의도된 컨트롤 루틴을 가집니다.

### 조작 매핑

| 입력 | 기능 | 상세 설명 |
|------|------|-----------|
| 마우스 호버 | 시선 방향 조정 | 커서 위치에 따라 개구리의 시선·점프 방향이 실시간 회전 |
| 마우스 우클릭 (Hold) + 드래그 | 점프 거리 조절 | 드래그한 거리에 비례해 점프 거리 조절 |
| 마우스 우클릭 (Release) | 점프 실행 | 설정된 거리·궤적·방향으로 점프 발사 |
| A 키 | 궤적 상승 | 점프 포물선을 높게 조정 (물고기 회피) |
| S 키 | 궤적 하강 | 점프 포물선을 낮게 조정 (새 회피) |
| 마우스 좌클릭 | 아이템 획득 (낼름) | 파리 등 곤충 아이템을 조준 후 좌클릭으로 섭취 |
| ESC | 일시정지 / 메뉴 | 게임 일시정지 및 옵션 메뉴 호출 |

### 점프 사이클

1. **조준 단계** — 마우스를 움직여 방향을 정합니다. 개구리의 시선이 실시간으로 따라갑니다. 한번 정해진 방향은 변경할 수 없습니다.
2. **거리 조절 단계** — 우클릭을 누른 채 드래그한 거리만큼 점프 거리가 늘어나며 착지 지점 궤적이 미리 보입니다.
3. **높이 조절 단계** — A/S 키로 점프 포물선의 높이를 미세 조정합니다.
4. **발사 단계** — 우클릭을 떼는 순간 모든 입력값이 확정되어 점프가 실행됩니다.

## 주요 콘텐츠

### 연잎 시스템

| 연잎 종류 | 특징 |
|-----------|------|
| 기본 연잎 | 특수 효과 없는 표준 발판. 안정적 콤보 유지의 기준점 |
| 삭은 연잎 | 밟은 직후 일정 시간이 지나면 파괴되어 가라앉음 |
| 미끄러운 연잎 | 착지 시 관성으로 미끄러져 다음 조준 위치가 어긋남 |
| 이동형 연잎 | 일정 패턴으로 좌우/상하 이동. 타이밍 점프 필요 |
| 회전 연잎 | 다음 점프 조준 시 방향이 강제로 소폭 회전됨 |
| 함정 연잎 | 식충식물 등 가시적 위협. 밟을 시 즉시 사망 |
| 탄성 연잎 (트램펄린) | 착지 후 다음 점프가 기본보다 훨씬 멀리 튕겨 나감 |
| 점멸 연잎 | 주기적으로 나타났다 사라짐. 타이밍 점프 필수 |

### 아이템 & 버프

- **사거리 증가**: 기본 점프 충전 속도 또는 최대 거리가 일시적으로 증가
- **생존 수영 (자력 복귀)**: 물에 빠지더라도 1회 한해 근처 연잎으로 헤엄쳐 복귀하는 세이브 쉴드
- **점수 배율 부스트**: 일정 시간 동안 획득 점수에 추가 배율 적용

### 적 & 방해 요소

- **물고기**: 수면 아래 대기. 점프 높이가 너무 낮으면 잡아먹힘 → **A키**로 회피
- **새**: 상공을 배회. 점프 높이가 너무 높으면 낚아채임 → **S키**로 회피
- **환경 장애물**: 나뭇가지·쓰레기 더미 등이 점프 궤적을 가로막음
- **악어**: 후면에서 쫓아와 촉박한 긴장감을 부여

### 동적 날씨

| 날씨 | 효과 |
|------|------|
| 안개 (Fog) | 가시거리가 제한되어 다음 연잎과 장애물을 미리 파악하기 어려움 |
| 강풍 (Strong Wind) | 점프 궤적이 휘어짐. 바람 방향·세기를 계산해 조준 보정 필요 |
| 호우 (Heavy Rain) | 모든 연잎이 일시적으로 '미끄러운 연잎'으로 변경 |
| 구름 (Cloud) | 이동하는 구름이 화면을 가려 간헐적으로 착지 지점이 보이지 않음 |

### 사운드 & 피드백

- 판정마다 차등화된 효과음 (Yarr! / Great / Not bad..)
- 악어가 가까워질수록 BGM 레이어가 추가되어 긴장감 상승
- '낼름' 효과음과 함께 파리를 흡입하는 짧고 경쾌한 모션
- 물에 빠질 때 코믹한 'plop' 사운드와 슬로우 모션 카메라 연출
